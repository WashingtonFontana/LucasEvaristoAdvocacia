Guia Rápido

Use o menu lateral para navegar entre as seções. Cada seção permite editar os textos correspondentes do site. As alterações são salvas em content.json e aplicadas em tempo real. Para trocar as imagens, acesse Imagens no menu.

Endpoints da API

GET /api/conteudo/publico — Conteúdo público (sem auth)
GET /api/conteudo — Todo o conteúdo
PUT /api/conteudo/hero — Editar Hero
PUT /api/conteudo/especialidades — Editar Especialidades
PUT /api/conteudo/perfil — Editar Perfil
PUT /api/conteudo/contato — Editar Contato
PUT /api/conteudo/footer — Editar Rodapé
POST /api/imagens/logo — Upload Logo
POST /api/imagens/perfil — Upload Foto Perfil
POST /api/imagens/background — Upload Background
POST /api/conteudo/reset — Restaurar padrão
